#### Task B: Analyze — Error Log (4 points)

**Error 1:**
- **(1) Error identified:** "Station A have a new monitor" — incorrect verb conjugation (should be HAS, not HAVE)
- **(2) Corrected sentence:** "Station A has a new monitor."
- **(3) Rule explanation:** "The subject 'Station A' is singular, so we use **HAS** instead of **HAVE**." (1 pt for complete analysis)

**Error 2:**
- **(1) Error identified:** "These keyboard is wireless" — subject-verb disagreement (plural subject with singular verb)
- **(2) Corrected sentence:** "These keyboards are wireless and modern." (or "This keyboard is wireless and modern.")
- **(3) Rule explanation:** "The subject 'keyboards' is plural, so we use **ARE** instead of **IS**." (1 pt for complete analysis)

**Error 3:**
- **(1) Error identified:** "This mouse have better sensors" — incorrect verb conjugation (SINGULAR subject, PLURAL verb)
- **(2) Corrected sentence:** "This mouse has better sensors."
- **(3) Rule explanation:** "The subject 'mouse' is singular, so we use **HAS** instead of **HAVE**." (1 pt for complete analysis)

**Error 4:**
- **(1) Error identified:** "That SSDs is 1 TB capacity" — subject-verb disagreement (plural subject with singular verb)
- **(2) Corrected sentence:** "Those SSDs are 1 TB capacity." (or "That SSD is 1 TB capacity.")
- **(3) Rule explanation:** "The subject 'SSDs' is plural, so we use **ARE** instead of **IS**." (1 pt for complete analysis)
