## VISIÓN DEL SISTEMA

Este documento consolida el sistema completo de **22 prompts maestros + 52 arquetipos de actividad** organizados en 4 fases circulares que automatizan el diseño, planeación, ejecución y evaluación de guías de aprendizaje de inglés como segunda lengua (ESP) para programas técnicos y tecnológicos del SENA.

```
         FASE 1: ANÁLISIS (PM-1.x)
        ↗         2 prompts          ↘
FASE 4: EVALUACIÓN                   FASE 2: PLANEACIÓN
  2 prompts                            10 prompts (47 arquetipos)
        ↖                            ↙
         FASE 3: EJECUCIÓN (PM-3.x)
               6 prompts (5 arquetipos en PM-3.5)
```

## PRINCIPIO DE RECICLAJE CIRCULAR DE INPUT

El principio más importante del sistema: **todo lo que se lee, se escribe; todo lo que se escucha, se habla; todo lo que se produce, se consolida.**

```
PM-1.2 (CURACIÓN DE MATERIAL AUTÉNTICO)
   │ 2 stories reales curadas: Story A + Story B
   │ Vocabulario, grammar targets y universo narrativo extraídos
   ▼
CONJUNTO A — ESCRITURA
PM-2.3 (READING) ──→ PM-2.4 (WRITING) ──→ PM-2.5 (LITERACY & VOCABULARY)
   │ Story A adaptada     Produce texto       Consolida vocabulario
   │ como Master Anchor    basado en el        y habilidades
   │ inyecta vocab +       modelo del          lectoescritoras
   │ grammar               reading
   │
   │ (vocabulario consolidado)
   ▼
CONJUNTO B — ORAL
PM-2.6 (LISTENING) ──→ PM-2.7 (PRONUNCIATION) ──→ PM-2.8 (SPEAKING)
   │ Story B adaptada      Entrena fonemas        Producción oral
   │ a formato oral         y prosodia de los      con chunks y
   │ recicla vocab +        términos clave         pronunciación
   │ grammar del
   │ Conjunto A
   │
   │ (output oral)
   ▼
CONJUNTO C — SISTEMAS DEL LENGUAJE
PM-2.9 (LANGUAGE FUNCTIONS) ──→ PM-2.10 (STRUCTURE USE & GRAMMAR)
   │ Activa el PARA QUÉ             Ancla el CÓMO
   │ de las funciones                en estructuras
   │ comunicativas                   gramaticales
   │
   ▼
PM-3.5 (FINAL MISSION)
   │ Integra TODO lo de Fase 3
   │ Produce: Desempeño (oral) + Producto (artefacto)
   ▼
PM-4.1 (EVALUACIÓN)
   │ Evalúa desempeño + producto con instrumentos alineados
   ▼
→ REGRESA A PM-1.1 ←
```

## MARCO TEÓRICO QUE SUSTENTA EL SISTEMA

### Enfoques Pedagógicos Integrados

| Enfoque | Cómo se Integra en el Sistema |
|---|---|
| **ESP** (English for Specific Purposes) | Marco principal: todo contenido lingüístico parte del entorno ocupacional del programa técnico |
| **TBLT** (Task-Based Language Teaching) | PM-2.8 (Speaking) y PM-3.5 (Final Mission): tareas con vacío de información y resultado comunicativo real |
| **SIOP** (Sheltered Instruction) | Doble objetivo (contenido + lengua) en cada sesión; andamiaje sistemático; 8 componentes integrados en PM-3.2 |
| **UbD** (Understanding by Design) | Diseño inverso: los RAPs (Etapa 1) definen las evidencias (Etapa 2) que definen las actividades (Etapa 3) |
| **HOTS** (Taxonomía de Bloom) | Progresión cognitiva en actividades de cada PM (Analyze → Evaluate → Create); escalamiento en Workbook (EXTEND) |
| **Flipped Learning** | PM-3.4 Sección C (PREPARE) pre-activa vocabulario y genera indagación antes de la siguiente sesión presencial |
| **CLIL** | Marco complementario cuando el instructor técnico co-enseña; enfoque dual lengua + contenido |
| **Lexical Approach** | Reciclaje Circular de Input: chunks se extraen del Reading, se escuchan en Listening, se practican en Vocabulary, se producen en Writing/Speaking |

### Marco Institucional SENA

| Elemento | Cómo se Integra |
|---|---|
| **FPI** (Formación Profesional Integral) | Mundo del trabajo + Mundo de la vida en cada actividad |
| **ABP** (Aprendizaje Basado en Proyectos) | El proyecto formativo articula las 6 macrotemáticas como fases de un proyecto real |
| **4 Fases de la Guía** | PM-2.1 (Reflexión) → PM-2.2 (Contextualización) → PM-2.3 a PM-2.10 (Apropiación/Transferencia) → PM-3.5 (Misión Final) |
| **Triada de Evidencias** | PM-4.1: Conocimiento (Quiz) + Desempeño (Checklist) + Producto (Rúbrica) |
| **4 Fuentes del Conocimiento** | Instructor (Playbook), Entorno (escenarios ESP), TIC (Canva/Google Classroom), Trabajo (simulaciones laborales) |
| **Mejora Continua** | PM-4.1§5 Feedback Loop → Actividad 21 del flujo operativo SENA |
