## CONTENIDO MARÍTIMO POR CAPÍTULO (Unidad 1 ejemplo)

### Chapter 1: REINFORCE — Label the Ship
```
Look at the picture. Write the correct word.

This is the ________. (bridge / deck / hull)
These are ________. (lifeboats / mooring lines / hatches)
That is the ________. (bow / stern / funnel)
The ________ is on the port side. (gangway / crane / buoy)
```

### Chapter 1: EXTEND — Describe Your Dream Ship
```
Imagine a ship of the future. Draw it and write 5 sentences.

1. This is a ________ ship.
2. The ________ is very big.
3. These are ________.
4. The ship has ________.
5. It is ________.
```

### Chapter 1: PREPARE — Vocabulary Preview
```
Before next session, learn these 5 words:

1. captain = the person who commands the ship
2. chief officer = the second in command
3. bosun = the boss of the deck workers
4. watch-keeping = working on a schedule to look after the ship
5. command = to give orders

Write each word 3 times and try to use it in a sentence.
```
