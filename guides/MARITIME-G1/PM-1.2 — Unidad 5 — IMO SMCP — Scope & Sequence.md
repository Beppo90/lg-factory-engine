## GUIDE DESIGN DNA

### Nombre conceptual:
**"Speak Like a Mariner"**

### Enfoque y conceptualización:
Esta unidad es la culminación del programa. El aprendiz aprende y practica las frases estandarizadas del IMO SMCP (Standard Marine Communication Phrases) — el "idioma oficial" de las comunicaciones marítimas internacionales. Las SMCP no son inglés libre: son frases fijas, con palabras exactas, que todos los marineros del mundo deben usar en situaciones de rutina y emergencia. Esta unidad integra TODO el vocabulario y las estructuras de las Unidades 1-4 en un sistema de comunicación estandarizado.

### Entendimientos perdurables:
1. El SMCP existe porque en el mar, un malentendido puede matar — las frases estandarizadas eliminan la ambigüedad y garantizan comprensión universal.
2. Las comunicaciones de socorro (MAYDAY, PAN PAN) siguen un protocolo rígido; improvisar no es una opción — la forma importa tanto como el contenido.
3. Todo lo aprendido en las Unidades 1-4 (partes del buque, roles, puerto, operaciones) converge en las SMCP — son la aplicación final de todo el vocabulario marítimo.

## CONTENT CORE

### Technical Topics:
1. SMCP — General (Standard phrases for requesting information, assistance, and action)
2. SMCP — Distress and Urgency (MAYDAY, PAN PAN, SECURITÉ procedures)
3. SMCP — Navigation and Maneuvering (course changes, speed, position reporting)
4. SMCP — Cargo and Port Operations (loading, unloading, berthing communications)
5. SMCP — Safety and Security (fire, flooding, man overboard, ISPS alerts)

### Communicative Functions:
- **Requesting information:** "What is your position?" / "Say again."
- **Requesting action:** "Alter course to starboard." / "Reduce speed."
- **Reporting situations:** "I am on fire." / "I have steering failure."
- **Acknowledging:** "Message received and understood." / "I will comply."
- **Distress calling:** "MAYDAY MAYDAY MAYDAY. This is [vessel name]."

### Language Functions in Context:
| Structure | SMCP Example |
|-----------|-------------|
| SMCP fixed phrases | "MAYDAY MAYDAY MAYDAY. This is [vessel name]." |
| Request for information | "What is your position?" / "What is your destination?" |
| Request for action | "Alter course to port." / "Reduce speed to 5 knots." |
| Reported commands | "The pilot instructed the helmsman to alter course." |
| Conditional (emergency) | "If the engine fails, we must anchor immediately." |
| Passive (reports) | "The vessel is reported to be on fire." |

## THE INTEGRATIVE TASK

### Nombre: "The Emergency Communication Drill"
El aprendiz participa en una simulación de comunicaciones de emergencia usando SMCP. El equipo recibe un escenario de emergencia (incendio, vía de agua, MOB) y debe ejecutar las comunicaciones estandarizadas. Produce un Emergency Communication Log escrito y realiza la simulación oral.

### Entregables:
- **Escrito:** Emergency Communication Log (registro cronológico de comunicaciones SMCP)
- **Oral:** Emergency Communication Drill (simulación de 5 minutos con roles: Captain, VTS, Rescue)

## CURACIÓN DE MATERIAL AUTÉNTICO

### Fuente 1 — Documento oficial
**Título:** "IMO Resolution A.918(22) — Standard Marine Communication Phrases"
**Fuente:** IMO official publication
**Género:** International regulatory document
**Relevancia:** El documento fuente del SMCP. Contiene TODAS las frases estandarizadas con contexto de uso. Es la fuente primaria de esta unidad.
**Vocabulario extraíble:** todas las frases SMCP + vocabulario de emergencia, navegación y puerto
**Grammar natural:** SMCP fixed phrases, passive voice para reportes
**HOTS level:** Apply / Analyze

### Fuente 2 — Caso de estudio / Reporte
**Título:** "Costa Concordia — Communication Failures and SMCP Violations"
**Fuente:** Marine accident investigation reports / IMO analysis
**Género:** Accident report / Case study
**Relevancia:** Análisis real de un desastre marítimo donde las comunicaciones SMCP no se siguieron. Demuestra la importancia de las frases estandarizadas de forma dramática.
**Vocabulario extraíble:** MAYDAY, abandon ship, distress, man overboard, course alteration, VTS, heading
**Grammar natural:** Conditionals para análisis hipotético + Reported commands
**HOTS level:** Evaluate / Create

### Fuente 3 — Transcripción de comunicaciones
**Título:** "Real VHF Communications — Port Entry and Emergency Scenarios"
**Fuente:** VTS recordings / maritime training simulations
**Género:** Communication transcript / Training simulation
**Relevancia:** Transcripciones reales de comunicaciones VHF usando SMCP. El material directo para practicar listening y speaking.
**Vocabulario extraíble:** channel 16, position report, speed, heading, pilot, tugboat, anchorage, VTS
**Grammar natural:** SMCP phrases en contexto oral real
**HOTS level:** Remember / Understand / Apply
