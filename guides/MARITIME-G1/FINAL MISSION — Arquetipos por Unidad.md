## UNIDAD 1: SHIP OVERVIEW — "The Vessel Inspection"

```
Unidad 1 — Elige arquetipo de Final Mission:

A) Professional Simulation — Inspección PSC: el aprendiz es el inspector, el compañero es el Chief Officer
B) Problem-Solution — El buque tiene un equipo de seguridad defectuoso: diagnosticar y reportar
C) Production — Crear un "Ship Identification Guide" ilustrado para nuevos marineros
D) Collaborative Project — Cada equipo documenta un tipo diferente de buque
E) Gamified — Escape Room: resolver acertijos sobre partes del buque para "escapar"
```

### Recomendación: **A) Professional Simulation**
**Componente oral:** El aprendiz (como inspector PSC) recorre el buque (diagrama) y pregunta al Chief Officer sobre partes y equipo de seguridad usando demonstratives y verb to be.
**Componente producto:** Completa un PSC Inspection Report con hallazgos.

## UNIDAD 2: THE CREW — "The Crew Briefing"

```
Unidad 2 — Elige arquetipo de Final Mission:

A) Professional Simulation — Nuevo marinero recibe briefing del Chief Officer sobre roles y rutinas
B) Problem-Solution — Un tripulante está enfermo: reorganizar turnos y rutinas
C) Production — Crear un "Crew Handbook" con roles y rutinas del buque
D) Collaborative Project — Cada equipo representa un departamento (Deck, Engine, Catering)
E) Gamified — "Who Am I?" game: describe duties sin decir el nombre, equipo adivina
```

### Recomendación: **D) Collaborative Project**
**Componente oral:** Cada equipo (por departamento) presenta los roles y rutinas de su departamento al "nuevo marinero" (el instructor u otro equipo).
**Componente producto:** Entregan un Crew Department Profile escrito con organigrama, rutinas y órdenes básicas.

## UNIDAD 3: PLACES IN PORT AND POSITION — "The Port Arrival"

```
Unidad 3 — Elige arquetipo de Final Mission:

A) Professional Simulation — Comunicación VTS: el aprendiz es el VTS Operator, el buque llega
B) Problem-Solution — El buque no puede encontrar su atraque: resolver la ubicación
C) Production — Crear un "Port Guide" del Puerto de Cartagena para tripulaciones
D) Collaborative Project — Cada equipo mapea una zona del puerto
E) Gamified — Treasure hunt: seguir instrucciones de atraque para encontrar el "tesoro"
```

### Recomendación: **A) Professional Simulation**
**Componente oral:** El aprendiz (como VTS Operator) guía al buque (compañero) desde el canal de navegación hasta el atraque usando Present Progressive y Tag Questions.
**Componente producto:** Completa un Port Arrival Report con secuencia de llegada.

## UNIDAD 4: IN PORT — "The Cargo Operations"

```
Unidad 4 — Elige arquetipo de Final Mission:

A) Professional Simulation — Chief Officer coordina carga con Stevedore Supervisor
B) Problem-Solution — Carga peligrosa mal documentada: resolver el problema
C) Production — Crear un "Cargo Operations Manual" simplificado
D) Collaborative Project — Cada equipo planifica la carga de un tipo de buque diferente
E) Gamified — "Load the Ship": juego de estiba con restricciones de peso y espacio
```

### Recomendación: **B) Problem-Solution**
**Componente oral:** El equipo recibe un escenario de problema de carga (carga peligrosa sin documentación correcta). Discuten soluciones y presentan al "Port Agent" usando quantifiers y modals.
**Componente producto:** Escriben un Corrective Action Report con pasos para resolver el problema.

## UNIDAD 5: IMO SMCP — "The Emergency Drill"

```
Unidad 5 — Elige arquetipo de Final Mission:

A) Professional Simulation — Simulacro de emergencia completa con roles y comunicaciones SMCP
B) Problem-Solution — Emergencia múltiple: fuego + vía de agua, priorizar comunicaciones
C) Production — Crear un "Emergency Communication Poster" con frases SMCP clave
D) Collaborative Project — Cada equipo prepara un escenario de emergencia diferente
E) Gamified — "Survival Radio": juego de comunicaciones bajo presión de tiempo
```

### Recomendación: **A) Professional Simulation**
**Componente oral:** Simulacro completo: un equipo ejecuta las comunicaciones SMCP de MAYDAY, otro responde como VTS/Rescue. Roles: Captain, VTS Operator, Rescue Coordinator.
**Componente producto:** Escriben un Emergency Communication Log cronológico con todas las frases SMCP usadas.
