## GRAMMAR CONSTRAINTS POR UNIDAD

La restricción gramatical es OBLIGATORIA — cada unidad tiene un set fijo:

### Unidad 1: Ship Overview
- **Demonstratives:** This is the bridge. / That is the engine room. / These are lifeboats. / Those are mooring lines.
- **Verb To Be:** The ship is a container vessel. / The hull is steel. / The crew are on deck.

### Unidad 2: The Crew
- **Present Simple:** The captain commands the vessel. / The bosun checks the deck every morning. / Engineers maintain the engines.
- **Imperative:** Report to the bridge. / Secure all hatches. / Check the safety equipment. / Follow the captain's orders.

### Unidad 3: Places in Port and Position
- **Present Progressive:** The vessel is approaching the berth. / The crew are mooring the ship. / The crane is loading containers.
- **Tag Questions:** The ship is alongside the quay, isn't it? / The gangway is down, isn't it? / They are unloading cargo, aren't they?

### Unidad 4: In Port
- **Quantifiers:** There is much oil in the tanks. / There are many containers on deck. / There is little time before departure. / There are few crew members on board.
- **Modals:** You must wear a hard hat. / You should check the manifest. / The crane can lift 50 tons. / Could you repeat the instruction?

### Unidad 5: IMO SMCP
- **SMCP Standard Phrases:** Request for information, assistance, and action using IMO standardized marine communication phrases.
- **Reported commands/orders:** The pilot instructed the helmsman to alter course.
- **Conditionals for emergencies:** If the engine fails, we must anchor immediately.

## FUENTES AUTÉNTICAS MARÍTIMAS

Las fuentes curadas deben provenir de:

| Fuente | Tipo | Ejemplo |
|--------|------|---------|
| Shipping industry news | Artículos | gCaptain, Marine Insight, The Maritime Executive |
| Port authority manuals | Documentos oficiales | Autoridad Portuaria de Cartagena, IMO publications |
| IMO documentation | Regulaciones | SOLAS, MARPOL, ISPS Code |
| Maritime training videos | Transcripciones | MCA, Maritime Training Academy |
| Vessel inspection reports | Reportes técnicos | Port State Control reports |
| Ship logs and communications | Géneros auténticos | Deck logs, radio communications, SMCP transcripts |

**NO usar:** ESL genérico, contenido de turismo, vocabulario de cruceros de placer.
