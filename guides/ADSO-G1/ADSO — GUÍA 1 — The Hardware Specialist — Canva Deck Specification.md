## DESIGN SYSTEM

| Element | Specification |
|---------|--------------|
| **Background** | White (#FFFFFF) for content slides, Navy (#1F3A5F) for session title slides |
| **Title font** | Calibri Bold, 40pt, Navy (#1F3A5F) — content slides |
| **Title font** | Calibri Bold, 48pt, White (#FFFFFF) — session title slides |
| **Body font** | Calibri Regular, 28pt, Dark Gray (#333333) |
| **Accent color 1** | Blue (#4472C4) — examples, highlights |
| **Accent color 2** | Green (#2E7D32) — correct answers, key grammar |
| **Error color** | Red (#C0392B) — bugs, incorrect versions |
| **Logo** | SENA logo, bottom right corner, all slides |
| **Max text per slide** | 7 lines (title + 6 content) |
| **Min font size** | 24pt (legible from back of classroom) |

#### SLIDE 9: AUDIO CUES
**Session:** 3
**Momento:** WHILE-A (Audio Cues intro)
**Actividad:** PM-2.4, Audio Cues
**Layout:** Content (3 bullet points)

**Content:**
```
🔊 BEFORE YOU LISTEN — 3 Things

1. "gig" = Gigabyte
   (4 gigs of RAM = 4 Gigabytes)

2. "lemme check" = "let me check"
   (very fast, very informal)

3. "is" vs "has" sound similar!
   is + ADJECTIVE (old, slow)
   has + THING (a problem, 4 GB)
```
