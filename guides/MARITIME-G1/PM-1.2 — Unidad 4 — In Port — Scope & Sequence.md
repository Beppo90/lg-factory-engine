## GUIDE DESIGN DNA

### Nombre conceptual:
**"Operations and Obligations"**

### Enfoque y conceptualización:
Esta unidad cubre las operaciones portuarias cotidianas: carga y descarga, documentación, servicios al buque, y las obligaciones de cada parte. El aprendiz desarrolla la capacidad de hablar sobre cantidades y cantidades de carga (Quantifiers) y de expresar obligación, posibilidad, habilidad y solicitud (Modals). Esta unidad es donde la comunicación marítima se vuelve más operativa — el buque está en puerto y se necesita coordinar con múltiples actores.

### Entendimientos perdurables:
1. La carga y descarga requiere comunicación precisa sobre cantidades — "much/little/a lot of" no son sinónimos y su uso incorrecto puede causar errores operativos.
2. Los modales (must/should/can/could) no son opcionales en inglés marítimo — expresan obligaciones legales, recomendaciones de seguridad, habilidades del buque y solicitudes corteses a autoridades.
3. La documentación portuaria (manifest, bill of lading, cargo declaration) es el lenguaje escrito del comercio marítimo — leerla y completarla en inglés es competencia laboral directa.

## CONTENT CORE

### Technical Topics:
1. Operaciones de carga: loading, unloading, stowage, lashing, dunnage, pallets, TEU (Twenty-foot Equivalent Unit)
2. Tipos de carga: bulk cargo, containerized cargo, liquid cargo, break-bulk cargo, hazardous cargo
3. Documentación portuaria: bill of lading, cargo manifest, customs declaration, port clearance, crew list
4. Servicios portuarios: pilotage, towage, bunkering, freshwater supply, waste disposal, provisioning
5. Obligaciones y permisos: port regulations, ISPS code requirements, safety obligations

### Communicative Functions:
- **Describing quantities:** "There is much oil in the tanks." / "There are many containers on deck."
- **Expressing obligation:** "You must wear a hard hat." / "All crew must attend the safety drill."
- **Making requests:** "Could you repeat the instruction?" / "Could we start loading?"
- **Giving advice:** "You should check the manifest." / "The captain should notify the port authority."
- **Stating ability/possibility:** "The crane can lift 50 tons." / "We can start loading at 0800."

### Language Functions in Context:
| Structure | Maritime Example |
|-----------|-----------------|
| much + uncountable noun | "There is much oil in the bunker tanks." |
| many + countable noun | "There are many containers on the quay." |
| little/few + noun | "There is little time before departure." / "There are few crew on board." |
| a lot of + both | "There is a lot of cargo to load." |
| must + V (obligation) | "You must wear safety equipment." |
| should + V (recommendation) | "You should check the cargo manifest." |
| can + V (ability/possibility) | "The vessel can carry 2,000 TEU." |
| could + V (request/polite) | "Could you send the pilot boat?" |

## THE INTEGRATIVE TASK

### Nombre: "The Cargo Operations Plan"
El aprendiz asume el rol de un Chief Officer que debe preparar un plan de carga para la próxima escala. Produce un Cargo Operations Plan escrito (qué cargar, dónde, cuánto, con qué equipo) y lo presenta oralmente al "Port Agent" justificando las decisiones.

### Entregables:
- **Escrito:** Cargo Operations Plan (tabla de carga + 6 oraciones con quantifiers y modals)
- **Oral:** Briefing al Port Agent (3 minutos, explicando el plan y respondiendo preguntas)

## CURACIÓN DE MATERIAL AUTÉNTICO

### Fuente 1 — Reporte de industria
**Título:** "Port of Cartagena Sets New Container Handling Record — 3.2 Million TEU"
**Fuente:** Port authority statistics / industry news
**Género:** Industry report / news article
**Relevancia:** Datos reales de operaciones portuarias en Cartagena. Contains quantifiers naturally (many containers, much cargo, a lot of TEU).
**Vocabulario extraíble:** TEU, containerized cargo, loading, unloading, gantry crane, terminal, bulk cargo, port clearance
**Grammar natural:** Quantifiers + Modals para capacidades del puerto
**HOTS level:** Analyze / Evaluate

### Fuente 2 — Documento oficial
**Título:** "ISPS Code — Port Facility Security Requirements"
**Fuente:** IMO ISPS Code documentation
**Género:** Regulatory document / security manual
**Relevancia:** Documento real sobre obligaciones de seguridad portuaria. Contiene modals obligatorios (must, shall) y vocabulario de documentación.
**Vocabulario extraíble:** ISPS code, port regulations, customs declaration, port clearance, hazardous cargo, crew list, cargo manifest
**Grammar natural:** Must/should para obligaciones y recomendaciones
**HOTS level:** Apply / Analyze

### Fuente 3 — Género auténtico
**Título:** "Sample Bill of Lading and Cargo Manifest — Container Ship Operations"
**Fuente:** Shipping company documentation / IMO model forms
**Género:** Official form / cargo document
**Relevancia:** Formularios reales de carga marítima. El aprendiz practica leer y completar documentos auténticos.
**Vocabulario extraíble:** bill of lading, cargo manifest, stowage, lashing, dunnage, hazardous cargo, TEU, bunkering, pilotage, towage
**Grammar natural:** Modals para permisos y obligaciones en contexto de documentación
**HOTS level:** Apply / Create
