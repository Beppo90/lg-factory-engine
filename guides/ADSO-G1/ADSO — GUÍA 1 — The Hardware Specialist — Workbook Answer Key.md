# CHAPTER 4: GRAMMAR DRILL

**Activity 1 — Fill in the Blank:**
1. The CPU **is** the brain of the computer.
2. My laptop **has** 16 GB of RAM.
3. **This** is an HDMI port.
4. The monitor **has** a VGA port.
5. **Those** are USB cables.
6. An SSD **is** faster than an HDD.
7. The keyboard **is** compatible with the new system.
8. **That** is a mechanical drive.
9. This workstation **has** only 4 GB of RAM.
10. **Those** are old USB 2.0 cables.

**Activity 2 — Syntax Bugs:**
1. ✗ → "The RAM **is** 16 gigabytes." (RAM is singular)
2. ✓ (Correct)
3. ✗ → "This laptop **has** an SSD." (singular → has)
4. ✗ → "Those **are** USB cables." (plural → are)
5. ✗ → "An SSD is **faster** than an HDD." (no "more" with -er)

**Activity 3 — Write Your Own:**
No answer key único. Evaluar:
- Sentence 1: ¿Usa IS correctamente?
- Sentence 2: ¿Usa HAS correctamente?
- Sentence 3: ¿Usa un demonstrative (THIS/THAT/THESE/THOSE) correctamente?
