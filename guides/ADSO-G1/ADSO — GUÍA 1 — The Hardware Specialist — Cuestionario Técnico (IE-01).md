## SECTION 5: GRAMMAR HOTS (10 pts)

| # | Answer | Points | Notes |
|---|--------|--------|-------|
| 19 | IS sentence | 1 | Ej: "PC #1 is slow." / "The CPU on PC #1 is an i3." Must use IS correctly + data from table |
| 20 | HAS sentence | 1 | Ej: "PC #2 has 8 GB of RAM." Must use HAS correctly + data from table |
| 21 | Comparison | 1 | Ej: "PC #2 is faster than PC #1." / "An SSD is faster than an HDD." |
| 22 | Error: "Those monitor is" | 1+1 | Correct: "Those monitors ARE" / "That monitor IS". Rule: Those = plural → are. That = singular → is. |
| 23 | Error: "more faster" | 1+1 | Correct: "An HDD is SLOWER than an SSD" / "An SSD is FASTER than an HDD." Rule: Faster already has "more" inside. No double comparative. |
| 24 | Justification with demonstratives | 1+1+1 | 1 for valid choice. 1 for logical justification. 1 for use of this/that/these/those. |

**Sample response for #24:**
> "I approve Department B because those monitors are not compatible with the new equipment. This is a smaller budget ($1,000 vs $2,500) and the problem affects everyone in the lab."
> **Score:** 1 (valid choice) + 1 (logical) + 1 (those + this) = 3/3
